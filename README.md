# Parking Manager System 🚗

## 📌 Description
This project is a smart parking management system that detects and manages parking spaces using computer vision and deep learning techniques.

## 🎯 Objective
The goal of this project is to automate parking space detection and improve parking efficiency by identifying available and occupied slots.

## ⚙️ Features
- Detect parking spaces using YOLO model  
- Identify available and occupied spots  
- Image-based parking analysis  
- Simple and efficient system  

## 🧠 Technologies Used
- Python  
- OpenCV  
- YOLOv8  
- Jupyter Notebook  

## 📁 Project Structure
- `parking-managment-yolo.ipynb` → Main model implementation  
- `f.py` → Supporting script  
- `img.png` → Sample image  
- `yolov8m.pt` → Pretrained model  

## ▶️ How to Run
1. Install required libraries  
2. Run the notebook file  
3. Load the model and test on an image  

## 📊 Results
The system can successfully detect parking spaces and determine whether they are occupied or free.
